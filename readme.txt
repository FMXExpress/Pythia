3. How to compile Pythia?
   I developed it using Dev-C++ 5 beta and g++ 3.4.2. The Dev-C++ project file is included.
   If you don't use Dev-C++ just compile all cpp files to one binary file.
   PeLib (http://www.pelib.com) is required to compile Pythia.
